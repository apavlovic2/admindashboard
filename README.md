# Admindashboard

Admin dashboard with a Home page, Payment orders page and Users page

## Development server

To start a local development server that runs the frontend, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`.

## JSON server

To start a local JSON server that runs the backend, run:

```bash
npx json-server --watch db.json --port 3000
```
